# DMA validation study

This repository contains code and data to reproduce the analysies for a mixed-methods validation study of a data maturity assessment in long-term care (LTC). This repository contains the neccessary files to reproduce the quantitative part of the study in which structural validation was tested using both confirmatory and exploratory factor analyses. Additionally, chronbach alpha values were calculated for the new domains. 

## Repository structure

- `scripts/` - R script for the EFA, CFA and chronbach alpha analysis
- `data/` - dataset used in the study
- `results/` - Figure generated by the scripts

## How to reproduce

1. Install R (version 4.3 or higher)
2. Install required R packages:
```r
install.packages(c"")
```
3. Run script

- `scripts/data_maturity_assessment_validation_full.R` -> runs all analyses
